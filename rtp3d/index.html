Background — what the client asked for

Client's own words (via project brief):

"For the most part, I want to take my current website and make it better, with an emphasis on improving navigation so customers know exactly where to go based on their needs."

Reference given: the3dprintstop.com — a similar business, as the direction the client wants to head in.

Requested improvements:

  1. Shop page — currently endless scrolling. Categorize products, improve flow.
  2. Homepage — completely revamp, potentially with new graphics.
  3. Content & Navigation — add a blog section, and dedicated pages per business line (3D Project Request, UV Printing, Laser Services, Printer Repair), each with its own request form.

3

Proposed scope

3.1 Platform & theme

Migrate to Shopify, built on the Ella theme (HaloThemes, sold via ThemeForest — already licensed, no additional cost to the client; 45,000+ merchants, 6 months support + lifetime updates included; not the official Shopify Theme Store).

Why Ella specifically: it ships with a genuinely built-in blog template (verified on the live Ella demo: sidebar with recent articles, tag filtering, featured-post layout, related products in-sidebar) — the client gets the blog section they asked for without paying for a custom build. Note transparently: Ella is a third-party ThemeForest theme, not distributed through Shopify's own Theme Store — support comes from HaloThemes, not Shopify. This is standard practice and low-risk given the theme's scale and track record, but worth the client knowing before purchase.

3.2 Homepage redesign

Rebuild with the same information (services, credibility, featured prints) but restructured for scannability instead of scroll-heavy portfolio galleries:

  • A "How can we help you today?" navigational card grid near the top (I have a file to print / Design something custom / UV printing & signage / Laser engraving & cutting / Printer repair / Shop ready-made prints) — routes visitors to the right page in one click instead of making them scroll and self-orient.
  • Smaller, optimized image sizes throughout (the current page's oversized project photos are the single biggest driver of excessive scrolling).
  • Condensed "What We Offer" / "Why Choose Us" sections — shorter copy blocks, not full paragraphs.
  • New graphics/photography where the client can supply them (see Open Questions — this can become a content-timeline dependency, same as any redesign that assumes new visual assets).

3.3 Shop / Collection page

Replicate the current shop page, moving from 2 products per row to 3–4 per row.

3.4 Product detail page

Replicate current PDP content (description, specs, variants).

3.5 Navigation & dedicated service pages

  • Keep the existing top-level structure (Shop, 3D Project Request, UV Printing, Laser Services, Printer Repair, Contact) but rebuild each service page with its own request form.
  • Recommended form app: Qikify Contact Form Builder — unlimited custom forms with a centralized submission dashboard (analytics, CSV export, email notifications), so all 3 service-request forms can run through one app and one inbox. Pricing: Free (1 form only — not enough for 3 pages) · Basic $5.99/mo (3 forms — matches exactly what's needed) · Premium $7.99/mo (unlimited forms + auto-response emails). Recommend Basic at minimum.
  • Open question for the client: does this centralized setup cover their needs, or is there existing CRM/email routing from the current Squarespace forms that should be preserved instead? Recommend resolving this before build starts.

3.6 Blog

Use Ella's built-in blog template (see 3.1) — no custom section build required. Preview: new-ella.myshopify.com/blogs/news

3.7 Instant Quote page

Implement using Filaquote (Shopify App Store listing: apps.shopify.com/3d-print-quote), which matches exactly what the client asked for. Verified capabilities:

  • Customers upload STL / 3MF / STP files and get an instant quote from geometry analysis — no manual pricing.
  • Native BambuStudio and PrusaSlicer 3MF support (relevant since RTP3D prints on Bambu Lab hardware per their own site copy).
  • Multi-file orders, per-file material selection, interactive 3D preview, abandoned-quote recovery emails, analytics dashboard.

Pricing tiers: Free (50 quotes/mo, "Powered by Filaquote" branding, English only) · Maker $15/mo (unlimited quotes, no branding, abandoned-quote recovery, 15 languages) · Business $49/mo (adds an order-approval queue). Recommend Maker at minimum for a professional storefront — the free tier's visible third-party branding isn't a good look on a paid service business, and abandoned-quote recovery is directly relevant to an instant-quote flow.

3.8 Header cleanup

Reduce header clutter: move social icons out of the primary nav row (into the announcement bar or footer), add a search icon (currently missing) and account icon alongside cart, and surface a persistent trust bar (location, phone, veteran-owned, shipping) plus a prominent "Request an Estimate" CTA — matching the pattern already validated on the reference mockup.

Open questions before finalizing

  1. Form management: recommended Qikify Contact Form (Basic tier, $5.99/mo) as one centralized app for all 3 service pages — confirm this covers their needs, or if existing CRM/email routing from Squarespace should be preserved instead.
  2. Filaquote tier: confirm Maker ($15/mo) vs. Business ($49/mo) vs. starting on Free and upgrading later.
  3. New homepage graphics: does the client have photography/graphics ready to supply, or does sourcing need to be scoped separately? (If new assets are needed, this is a content-production dependency outside dev hours — same pattern as any redesign that assumes new visual assets.)
  4. Ella theme sourcing: theme is already licensed via ThemeForest/HaloThemes (no additional cost to the client) — confirm the client is comfortable with a ThemeForest-sourced theme rather than one from Shopify's own Theme Store; functionally identical once installed, but the support channel differs.
  5. Redirects / SEO continuity: current Squarespace URLs (e.g. /shop/p/..., /projectrequest) will change structure on Shopify. Recommend a URL redirect map as part of launch QA to preserve any existing search rankings/backlinks — not included in the 75h above unless the client wants it scoped now.
