Current Site Overview

WooCommerce store for an Australian retailer specializing in animal behavior/training equipment — remote trainers (DT Systems, PetSafe, SportDog), bark collars, in-ground containment fences, parts & accessories, pet accessories, livestock equipment (cattle immobilizers), food & water products, and training whistles. The homepage banner currently has no clickable CTA, product cards use plain text links instead of icon buttons, collection pages have no filters, product thumbnails need a slider treatment, and the minicart lacks quantity controls — see Section 4 for the full review; these fixes are already folded into Option A's scope below.

Two Implementation Directions

Two directions are quoted below so the client can choose.

Option A

Replicate

Clone the current site's layout and structure (navigation, homepage sections, category/product page layout, footer) onto Shopify using an existing/free Shopify theme's default layout as the build foundation; migrate products, categories, and site content from WooCommerce to Shopify. This scope already includes implementing the team's recommended UX/design fixes listed in Section 4 (banner CTA, icon-based add-to-cart, collection filters, thumbnail slider, minicart quantity control) — this is our own recommended approach to a proper replicate, not a bare 1:1 clone of the current site's rough edges.

Option B

New Design (based on the theme's default layout)

Built on the recommended theme's own default layout — not a custom layout. Instead of forcing the current custom design onto Shopify, this direction starts from a well-matched premium theme's out-of-the-box layout and customizes it to the client's branding (logo, colors, fonts, homepage sections), rather than rebuilding the current page structure. This route typically lands cheaper than Option A, because several of the same fixes (icon-based add-to-cart, product filters, thumbnail slider, cart-drawer quantity control) come built into the theme's default layout rather than needing custom development.

Recommended theme: Impulse (Archetype Themes, premium license) — same recommendation as for The Farmer Shop, for consistency if the agency wants both sites on a shared theme foundation, and because it fits this catalog well too: built-in product filters, image galleries (covers the thumbnail-slider ask), a cart drawer with quantity controls, and a clean mega-menu structure that can also host the content-heavy nav items (Articles, FAQs, Blog). Since theme cost isn't a constraint for the client, this is the strongest fit rather than a free/basic theme that would need custom work to add the same features.

Design & UX Fixes Included in Option A

These are the specific fixes referenced in Section 2 above — already part of Option A's 55h scope, and mostly covered natively by Option B's recommended theme (flagged above where they aren't).

1. Homepage banner — no clickable CTA

The homepage banner (sheep + working dog photo, "Dog Training Collars — Expert Advice — Parts & Repair") currently has no button or link — it's a static image with text overlay only. Recommended: add a button or make the banner itself clickable through to the relevant collection.

2. Product card — text links instead of icon buttons

Product cards currently show "Add to Cart | Show Details" as plain text links. Recommended: convert "Add to Cart" into an icon-based button, consistent with the more modern card treatment recommended for The Farmer Shop.

3. Shop/collection filters

Same gap as The Farmer Shop: collection pages have no filtering. Recommended: add the same filter treatment (sidebar or above-list, collapsing to a filter icon + popup on mobile).

4. Product page — thumbnail gallery → single-row slider

On product pages (e.g. the Kurgo dog water bowl, SKU K018), the thumbnail gallery currently wraps into a 2-row grid (6 thumbnails as 3 columns × 2 rows) below the main image. Recommended: convert this into a single-row horizontal slider so all thumbnails sit on one line and scroll/swipe instead of stacking — more compact and more mobile-friendly.

5. Minicart — quantity control

Same gap as The Farmer Shop: the minicart shows item, price, and a remove ("×") option, but no way to adjust quantity inline — shoppers must go to the full cart page. Recommended: add inline quantity +/- controls, same treatment as recommended for The Farmer Shop (including the reference example reviewed there).
