Implementation Notes

  • Contact Us: uses Shopify's own native contact form by default, extended with the design's extra fields. Qikify (see Apps table above) is only needed if app-managed forms are wanted instead.
  • Search: layout-only customization — the search function itself (indexing, ranking) stays the theme's default.
  • Notice: built as a Blog category, not a separate template — Notice posts are regular blog articles grouped into their own category.
  • PDP: metafields/metaobjects work if all products share the same section layout. Products needing a genuinely different layout would need separate product templates — to confirm before build.
  • About: built as a standard page template, not metafields/metaobjects — it's a one-off page, not repeated structured content.
