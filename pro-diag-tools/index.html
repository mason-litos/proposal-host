Current Site Assessment

We reviewed the live Shopify store (prodiagtools.com) prior to scoping this proposal.

Strengths

  • Technical, trustworthy color palette — the tones used across the site fit a technical/diagnostic-tools brand and read as credible.
  • Strong product visuals — most listings have clear photography of the physical hardware or clean screenshots of the diagnostic software.
  • Clear category structure — products are grouped in a way that helps shoppers quickly locate the right product line.

Weaknesses

  • Cluttered layout — the homepage and collection pages are visually dense; white space between content blocks isn't optimized, which reads as crowded.
  • Weak typographic hierarchy — heading sizes don't contrast enough against body text, so pages feel flat and harder to scan.
  • Limited shopping/checkout experience:
    • No Ajax/predictive quick search — a baseline expectation on modern storefronts.
    • No mini-cart (side cart). Without it, every "Add to cart" click either redirects to the full cart page or leaves the shopper unsure their item was added — a broken flow that adds 1–2 unnecessary steps for anyone browsing multiple products in one session.

Recommended Design & UX Improvements

The following were identified during the review and are recommended for the rebuild. These are not included in the base 55h replication estimate (Section 4) — they're priced separately as an optional Phase 2 so the client can decide whether to include them.

A. Header & branding

Currently the header is split into two visually disconnected bars: a white top bar (logo, search, contact info, cart) sitting above a separate dark-navy navigation bar. Recommended: unify both into a single navy-colored header block so the logo, search, and nav read as one cohesive brand element instead of two mismatched sections — cleaner look, better logo visibility, same functional layout (search, phone/email, cart) preserved.

B. Homepage banner

On the current banner, the "Shop Now" button sits low-contrast against the hero photo and easily gets lost, and its relationship to the brand-shortcut tabs (BMW/VW/VAG/etc.) and the trust-badge strip below (120+, 4.9 rating, Windows, Fast Email, Secure) isn't visually clear — everything reads as scattered rather than one composed banner. Recommended: redesign the banner so the CTA button is a deliberate part of the image composition (not just overlaid), keep it clickable through to the relevant collection, and tighten the visual relationship between headline, CTA, brand shortcuts, and trust badges.

C. Support/service section discoverability

The site sells both products and technical support/services, so the support section needs its own distinct visual treatment (separate from the product category cards) so shoppers can find it at a glance instead of it blending into the catalog. Recommended treatment: an editorial-style, alternating image/text layout — large photography paired with a bold headline, short subtitle, description copy, and a "Learn More" link — one block per service (e.g. "Customizations with One-Click Apps," "Diagnostics").

D. Mini-cart (side cart) + Ajax quick search

Add a slide-out side cart that shows the item just added, running total, and a prominent "Checkout" button, with the ability to dismiss it and keep browsing without losing their place. Pair this with predictive/Ajax search in the header for both desktop and mobile.

E. Product card & product page polish

  • Hide the star-rating display on products that have no reviews yet, instead of showing an empty/zero rating.
  • Restyle the "Add to cart" button (solid background treatment) for better visibility on the product card.
  • On hover, swap the visible action to "Add to cart" + a new Quick View button, so shoppers can preview a product without leaving the collection page.

F. Footer redesign

Restructure the footer layout for better organization and visual balance.

G. Responsive fixes

Correct the mobile header/logo misalignment currently present on the live site.

H. Collection filters

The sidebar filter UI on the current site is present but buggy/non-functional; the WooCommerce rebuild needs full filter parity (all the filter facets already defined on the current site) working correctly out of the box.

Recommended Theme & Plugins

Theme options

Final choice depends on client budget — to be confirmed.

  • Premium theme (e.g. Woodmart or similar WooCommerce theme) — faster to configure, more built-in flexibility, license purchased directly by the client.
  • Free theme + custom development — a free WooCommerce-compatible theme customized with additional code to deliver the same layout and features. This route needs more custom-code time than a premium theme, since more of the design/functionality has to be built rather than configured — factor this into the estimate if the client chooses this path.

Recommended plugins

To bring the WooCommerce build to functional parity with the current Shopify store.

  • Contact Form 7 — contact/support forms.
  • WP SMTP — only needed if the hosting provider doesn't already support outbound mail delivery.
  • Elementor — page builder, used for custom sections (banner, support section, footer) that go beyond the default theme layout.
  • Product Reviews plugin — to replace the review functionality currently running on Shopify.
Note: license/subscription costs for any premium theme or premium plugin tier are the client's responsibility and are not included in the quote below.
